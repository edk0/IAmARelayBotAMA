## things

- `client.js` - Minecraft client. invoke as `node client.js <server>`
- `ircbot.py` - Redis client -> IRC bot
- `websocket-server.py` - Redis client -> IRC server
- `tx_redis.py` - Redis protocol implementation for Twisted
